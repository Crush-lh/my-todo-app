# 待办清单 (Todo App)

基于 **Next.js 15** + **Supabase** + **Server Actions** 构建的待办清单应用。

## 功能

- 添加待办事项（支持描述）
- 标记完成/未完成
- 编辑待办内容
- 删除待办
- 实时统计

## 技术栈

- Next.js 15 (App Router)
- React 19
- TypeScript
- Supabase (PostgreSQL)
- Server Actions (无需 API 路由)

## 部署到 Vercel

### 1. 创建数据库表

登录 [Supabase Dashboard](https://app.supabase.io)，在 SQL Editor 执行：

```sql
create table todos (
  id bigint generated by default as identity primary key,
  title text not null,
  description text,
  completed boolean not null default false,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  updated_at timestamp with time zone default timezone('utc'::text, now())
);

-- 启用 RLS（可选，如果需要认证）
alter table todos enable row level security;

-- 允许匿名访问（仅适合个人/测试项目）
create policy "Allow anonymous access" on todos
  for all using (true) with check (true);
```

## 特点

- **Server Actions** — 无需写 REST API，直接调用服务端函数
- **ISR / Revalidate** — 数据变更后自动刷新页面
- **零 API 路由** — 所有数据库操作都在 Server Actions 中完成
- **响应式设计** — 移动端友好
